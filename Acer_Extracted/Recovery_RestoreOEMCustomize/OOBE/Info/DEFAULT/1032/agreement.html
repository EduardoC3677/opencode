ΕΚΧΩΡΗΣΗ ΑΔΕΙΑΣ

1. Να εγκαταστήσετε και χρησιμοποιήσετε το Λογισμικό μόνο σε ένα συγκεκριμένο ηλεκτρονικό υπολογιστή. Για κάθε ηλεκτρονικό υπολογιστή στον οποίο θα χρησιμοποιηθεί το Λογισμικό απαιτείται ξεχωριστή Άδεια.

2. Να δημιουργήσετε μόνο ένα αντίγραφο του Λογισμικού αποκλειστικά ως αντίγραφο ασφαλείας ή για τη τήρηση αρχείου.

3. Να διατηρήσετε σε έντυπη μορφή ένα αντίγραφο οιωνδήποτε ηλεκτρονικών εγγράφων περιλαμβάνονται στο Λογισμικό, υπό τον όρο ότι λάβατε τα έγγραφα σε ηλεκτρονική μορφή.

8. Δεν σας ζητάμε να παρέχετε ευαίσθητες προσωπικές πληροφορίες (όπως πληροφορίες σχετικά με την υγεία σας, τις πολιτικές ή θρησκευτικές σας πεποιθήσεις, τα ποινικά αδικήματα που καταγγέλλονται ή έχουν διαπραχθεί ή τον σεξουαλικό σας προσανατολισμό) και σας ζητάμε να μην μας παρέχετε τέτοιες πληροφορίες. Οποιαδήποτε τέτοια πληροφορία παρέχετε οικειοθελώς σε εμάς θα χρησιμοποιείται μόνο σύμφωνα με την παρούσα Πολιτική προστασίας προσωπικών δεδομένων.

• AcerID. Για να σας δώσουμε τη δυνατότητα να δημιουργήσετε ένα AcerID, πρέπει να συλλέξουμε και να επεξεργαστούμε ορισμένες ή όλες τις παρακάτω πληροφορίες: διεύθυνση ηλεκτρονικού ταχυδρομείου, όνομα, επώνυμο, αναγνωριστικό σύνδεσης σε μέσα κοινωνικής δικτύωσης (όπως αναγνωριστικό Facebook ή αναγνωριστικό Google), χώρα και γλώσσα. Αυτές είναι οι ελάχιστες και, συνεπώς, απαραίτητες πληροφορίες που χρειαζόμαστε, για να σας δώσουμε τη δυνατότητα να δημιουργήσετε ένα AcerID. Κατά τη δημιουργία του AcerID, ενδέχεται επίσης να συμπληρώσετε το προφίλ σας παρέχοντας και άλλες πληροφορίες, όπως τον αριθμό τηλεφώνου σας, τη διεύθυνσή σας, πληροφορίες που χρειάζονται για την εγγραφή ενός προϊόντος Acer κ.λπ. Επιπλέον, μπορείτε να διαχειριστείτε το προσωπικό προφίλ σας επιλέγοντας να λαμβάνετε το ενημερωτικό δελτίο της Acer και άλλα μηνύματα, από τα οποία μπορείτε να απεγγραφείτε οποιαδήποτε στιγμή.

Ενημερωτικό δελτίο Acer. Μπορείτε επίσης να επιλέξετε να λαμβάνετε το ενημερωτικό μας δελτίο χωρίς να δημιουργήσετε ένα AcerID. Σε αυτήν την περίπτωση, οι μόνες πληροφορίες που χρειαζόμαστε για συλλογή και επεξεργασία είναι η διεύθυνση ηλεκτρονικού ταχυδρομείου, η χώρα και η γλώσσα σας, οι οποίες είναι απαραίτητες για την αποστολή του ενημερωτικού δελτίου μας.

3. Τι κάνουμε με τις προσωπικές σας πληροφορίες και σε ποιον τις κοινοποιούμε:

Οι προσωπικές πληροφορίες ενδέχεται να χρησιμοποιηθούν για την υποστήριξη της σχέσης σας με την Acer ως πελάτη της. Θέλουμε να χρησιμοποιούμε αυτά τα δεδομένα για να προσαρμόζουμε την εμπειρία σας στον ιστότοπό μας και εν γένει μαζί μας. Δεν συλλέγουμε ή χρησιμοποιούμε τις προσωπικές πληροφορίες που υποβάλλετε για άσχετους σκοπούς, εκτός από τις περιπτώσεις όπου επιβάλλεται ή επιτρέπεται από τους τοπικούς νόμους να το πράξουμε χωρίς να προηγούμενη ενημέρωσή σας ή χωρίς τη λήψη ρητής συγκατάθεσής σας.

Η συλλογή και, στη συνέχεια, η επεξεργασία των προσωπικών πληροφοριών σας γίνεται για τους παρακάτω σκοπούς:

• Για να ανταποκριθούμε σε συγκεκριμένα αιτήματά σας, π.χ. για να σας δώσουμε τη δυνατότητα να δημιουργήσετε ένα AcerID και για να σας παρέχουμε πρόσβαση σε ένα σύνολο σχετικών υπηρεσιών (όπως η εγγραφή μιας συσκευής Acer, η δημιουργία του δικού σας κόσμου στο cloud με τις εφαρμογές abApps και η λήψη υποστήριξης). Η επεξεργασία των προσωπικών δεδομένων σας για αυτόν το σκοπό δεν απαιτεί την προηγούμενη συγκατάθεσή σας.

• Για σκοπούς μάρκετινγκ. Εκτός εάν μας δώσετε διαφορετική οδηγία (όπως περιγράφεται παρακάτω), θα σας αποστέλλουμε πληροφορίες σχετικά με τις διάφορες αναβαθμίσεις προϊόντων, τις ειδικές προσφορές και άλλα νέα προϊόντα και υπηρεσίες της Acer που πιστεύουμε ότι ενδέχεται να σας ενδιαφέρουν. Μόνο η Acer (ή οι αντιπρόσωποι οι οποίοι εργάζονται για λογαριασμό της Acer) θα σας αποστέλλουν αυτά τα απευθείας μηνύματα. Αν δεν θέλετε να λαμβάνετε τέτοιου είδους μηνύματα, απλώς ενημερώστε μας κατά την παροχή των προσωπικών σας πληροφοριών. Επίσης, μπορείτε οποιαδήποτε στιγμή να επιλέξετε να μην λαμβάνετε πλέον μηνύματα, επικοινωνώντας με την Acer. Ανατρέξτε στην ενότητα της εξαίρεσης παρακάτω.

Εφόσον απαιτείται από το εφαρμοστέο Δίκαιο, θα σας αποστέλλουμε αυτά τα απευθείας μηνύματα μόνο εφόσον δώσετε τη ρητή συγκατάθεσή σας (προαιρετικά). Εάν έχετε δώσει τη συγκατάθεσή σας, αλλά δεν επιθυμείτε πλέον να λαμβάνετε τέτοιου είδους μηνύματα, απλώς ενημερώστε μας σχετικά όταν λάβετε το ενημερωτικό δελτίο, ακολουθώντας τη "Διαδικασία απεγγραφής" που υπάρχει σε αυτό. Μπορείτε επίσης να αρνηθείτε να λαμβάνετε μελλοντικά μηνύματα ανά πάσα στιγμή, επικοινωνώντας με την Acer. Λάβετε υπόψη ότι η ανάκληση της συγκατάθεσής σας δεν επηρεάζει τη νομιμότητα της επεξεργασίας με βάση τη συγκατάθεσή σας πριν από την ανάκλησή της. Στον βαθμό που επιτρέπεται από τη νομοθεσία, οι πληροφορίες τις οποίες παρέχετε ενδέχεται να προσπελαστούν, να υποβληθούν σε επεξεργασία ή να αποθηκευτούν από ή να μεταβιβαστούν σε οποιοδήποτε μέλος του ομίλου εταιρειών της Acer (ή στους υπεργολάβους τους) σε περιοχές σε όλο τον κόσμο, συμπεριλαμβανομένων εκείνων που βρίσκονται σε χώρες στις οποίες δεν διαμένετε. Σε αυτές τις περιοχές συμπεριλαμβάνονται χώρες οι οποίες δεν εξασφαλίζουν επαρκή προστασία των προσωπικών πληροφοριών. Ένας κατάλογος των οντοτήτων της Acer (και τις επίσημες έδρες τους) οι οποίες μπορούν να λαμβάνουν τα στοιχεία σας σε αυτές τις περιοχές είναι διαθέσιμος στη διεύθυνση www.acer-group.com/ag/en/TW/content/office-list. Η χρήση αυτή ενδέχεται να συνεχιστεί ακόμα και αν παύσουν να αποτελούν μέλη του ομίλου εταιρειών Acer. Όλες οι εταιρείες της Acer οι οποίες λαμβάνουν τις προσωπικές σας πληροφορίες θα τις χρησιμοποιούν σύμφωνα με την παρούσα Πολιτική προστασίας προσωπικών δεδομένων.

Η Acer ενδέχεται να κοινοποιήσει τις προσωπικές σας πληροφορίες σε έναν εξωτερικό οργανισμό για τη χρήση τους σε δραστηριότητες μάρκετινγκ ή προσέλκυσης πελατών, για (εξατομικευμένες) διαφημίσεις ή για τη βελτίωση ή ανάπτυξη αυτών των δραστηριοτήτων.

Οι πληροφορίες σας ενδέχεται να κοινοποιηθούν σε αντιπροσώπους ή αναδόχους της Acer με σκοπό την παροχή υπηρεσιών, την επεξεργασία συναλλαγών ή την εκτέλεση στατιστικής ανάλυσης για μια εταιρεία της Acer. Για παράδειγμα, εάν χρειαστεί να σας αποστείλουμε κάτι, πρέπει να κοινοποιήσουμε το όνομα και τη διεύθυνσή σας σε μια εταιρεία αποστολής. Παρέχουμε σε τρίτους αντιπροσώπους ή αναδόχους μόνο τον ελάχιστο αριθμό πληροφοριών που απαιτούνται για την ολοκλήρωση των ζητούμενων υπηρεσιών ή συναλλαγών.

Σε ορισμένες περιορισμένες περιπτώσεις, οι πληροφορίες σας ενδέχεται επίσης να αποκαλυφθούν σε τρίτα μέρη, όπου απαιτείται από τον νόμο ή έχουμε το δικαίωμα να το πράξουμε, για παράδειγμα ως απάντηση σε αίτημα κυβερνητικής ή ρυθμιστικής αρχής ή σε σχέση με πιθανές νομικές διαδικασίες. Η Acer δεν ευθύνεται για τις ενέργειες οποιασδήποτε κυβερνητικής ή ρυθμιστικής αρχής σε σχέση με τις πληροφορίες ή τα δεδομένα σας.

Μπορούμε, επίσης, να χρησιμοποιήσουμε τις πληροφορίες σας προς συμμόρφωση με τους νόμους, προς υπεράσπισή μας σε αξιώσεις στο πλαίσιο αυτών των νόμων ή όποτε χρειαστεί να ασκήσουμε τα νόμιμα επιχειρηματικά συμφέροντα και τις ανάγκες της Acer, στην οποία περίπτωση λαμβάνεται ειδική μέριμνα για τη διασφάλιση των δικαιωμάτων σας και την εξασφάλιση ότι τυχόν χρήση των προσωπικών πληροφοριών σας είναι δίκαιη, νόμιμη και ανάλογη.

Εάν συγχωνευθούμε ή οποιοδήποτε τμήμα του ομίλου Acer εξαγοραστεί από άλλη επιχείρηση (ή εάν ξεκινήσουμε συζητήσεις σχετικά με αυτό), ενδέχεται να μοιραστούμε τις πληροφορίες σας με μια τέτοια επιχείρηση και τους συμβούλους της. Ωστόσο, σε αυτήν την περίπτωση, θα διασφαλίσουμε ότι θα ληφθούν μέτρα ώστε τα δικαιώματά σας σε σχέση με τις πληροφορίες σας να συνεχίσουν να προστατεύονται.

4. Οδηγίες για παιδιά:

Η Acer ΔΕΝ θα συλλέγει εν γνώσει της πληροφορίες επικοινωνίας μέσω διαδικτύου από παιδιά ή ανηλίκους (όπως ορίζονται από τις τοπικές απαιτήσεις). Εάν διαπιστώσουμε ότι διαθέτουμε προσωπικές πληροφορίες για κάποιο παιδί ή ανήλικο, θα τις διαγράψουμε από τα συστήματά μας. Η Acer ενθαρρύνει τους γονείς ή τους κηδεμόνες (συμπεριλαμβανομένων άλλων νομικών εκπροσώπων) να περνούν χρόνο με τα παιδιά τους στο διαδίκτυο και να συμμετέχουν στις διαδικτυακές τους δραστηριότητες. Τα παιδιά ή οι ανήλικοι δικαιούνται να χρησιμοποιούν τις υπηρεσίες συναλλαγών διαδικτυακών πωλήσεων ΜΌΝΟ με τη συγκατάθεση του γονέα ή του κηδεμόνα τους και την πληρωμή για τη συναλλαγή ή στον βαθμό που επιτρέπεται από την ισχύουσα νομοθεσία. Αν είστε παιδί ή ανήλικος, πρέπει να λάβετε την άδεια των γονέων ή των κηδεμόνων σας για να ολοκληρώσετε συγκεκριμένες συναλλαγές σε αυτόν τον ιστότοπο.

5. Χρήση και αποκάλυψη ανώνυμων συγκεντρωτικών πληροφοριών:

Η Acer ενδέχεται επίσης να συλλέγει, να συγκεντρώνει και να διατηρεί ανώνυμες πληροφορίες σχετικά με τους πελάτες και τους επισκέπτες του ιστοτόπου μας, όπως χόμπι, ενδιαφέροντα, αγοραστικές συνήθειες και άλλα παρόμοια στοιχεία. Στη συνέχεια, αυτά τα δεδομένα μπορούν να χρησιμοποιηθούν για την προσαρμογή του περιεχομένου και των διαφημίσεών μας, ώστε να προσφέρουν στους επισκέπτες μας μια καλύτερη εμπειρία. Επιπλέον, η Acer ενδέχεται να κοινοποιήσει αυτές τις συγκεντρωτικές πληροφορίες σχετικά με τους επισκέπτες μας σε διαφημιστές, επιχειρηματικούς συνεργάτες, χορηγούς και άλλα τρίτα μέρη, για παράδειγμα, για να τους ενημερώσει σχετικά με το ποσοστό των ανδρών/γυναικών επισκεπτών ή το ποσοστό των επισκεπτών ενός συγκεκριμένου ηλικιακού εύρους.

Τα προσωπικά σας δεδομένα ή/και προφίλ δεν επιτρέπεται να ενοικιάζονται και να πωλούνται σε τρίτα μέρη χωρίς την προηγούμενη ρητή συγκατάθεσή σας.

6. Μέτρα ασφαλείας:

Προστατεύουμε τις προσωπικές πληροφορίες τις οποίες μας κοινοποιείτε από καταστροφή, απώλεια, αλλοίωση, μη εξουσιοδοτημένη αποκάλυψη ή πρόσβαση σε προσωπικά δεδομένα που μεταδίδονται, αποθηκεύονται ή υποβάλλονται σε άλλη επεξεργασία. Στην Acer, η πρόσβαση στις προσωπικές σας πληροφορίες περιορίζεται στα άτομα που τις χρειάζονται για τη δουλειά τους. Σύμφωνα με την πολιτική μας, διατηρούμε την ακεραιότητα των προσωπικών πληροφοριών τις οποίες έχουμε στην κατοχή μας. Για την αποτροπή παραβιάσεων της ασφάλειας, η Acer χρησιμοποιεί διάφορες τεχνικές ασφάλειας, όπως SET ή τυπική κρυπτογράφηση Secure Socket Layer (SSL) για τη μετάδοση πληροφοριών πιστωτικής κάρτας ή όπου απαιτούνται προσωπικές πληροφορίες, για την προστασία τυχόν προσωπικών πληροφοριών από μη εξουσιοδοτημένη πρόσβαση από χρήστες. Ωστόσο, δεν μπορούμε να εγγυηθούμε και πράγματι δεν εγγυόμαστε την ασφάλεια στο διαδίκτυο.

Επιπλέον, προσπαθούμε να διασφαλίσουμε ότι τα προσωπικά σας δεδομένα παραμένουν ακριβή και ενημερωμένα. Για τον λόγο αυτό, σας ζητούμε να μας ενημερώσετε για τυχόν αλλαγές στα προσωπικά σας δεδομένα (όπως μια αλλαγή στα στοιχεία επικοινωνίας σας).

7. Απορίες ή ερωτήσεις - Τα δικαιώματά σας:

Σε σχέση με την επεξεργασία των προσωπικών δεδομένων σας, έχετε ορισμένα δικαιώματα:

• Έχετε το δικαίωμα να λάβετε επιβεβαίωση για το εάν υποβάλλονται ή όχι σε επεξεργασία τα δεδομένα προσωπικού χαρακτήρα σας καθώς και να ενημερωθείτε για το ποια τέτοιου είδους δεδομένα σας υποβάλλονται σε επεξεργασία.

• Έχετε το δικαίωμα να ενημερώνεστε για (i) την προέλευση των δεδομένων προσωπικού χαρακτήρα σας, (ii) τους σκοπούς και τις μεθόδους της επεξεργασίας, (iii) την ταυτότητα του θεματοφύλακα δεδομένων και (iv) τις οντότητες ή τις κατηγορίες οντοτήτων τρίτων στις οποίες ενδέχεται να κοινοποιηθούν τα προσωπικά δεδομένα.

• Έχετε το δικαίωμα να αποκτήσετε: (i) ενημέρωση, διόρθωση ή ενσωμάτωση των δεδομένων σας και (ii) διαγραφή, ανωνυμοποίηση ή αποκλεισμό των δεδομένων σας, εάν αυτά υποβληθούν σε παράνομη επεξεργασία.

• Έχετε το δικαίωμα να αντιταχθείτε, εν όλω ή εν μέρει (i) για νόμιμους λόγους, στην επεξεργασία των προσωπικών δεδομένων σας, ακόμη και αν τα δεδομένα είναι σχετικά με τον σκοπό της συλλογής και (ii) στην επεξεργασία των δεδομένων προσωπικού χαρακτήρα σας, όπου αυτά υποβάλλονται με σκοπό την αποστολή διαφημιστικού υλικού ή την απευθείας πώληση, ή για την εκτέλεση ερευνών αγοράς ή εμπορικών επικοινωνιών.

Εάν έχετε ερωτήσεις ή απορίες σχετικά με τις πολιτικές και τις πρακτικές της Acer για τον χειρισμό προσωπικών πληροφοριών που λαμβάνονται μέσω αυτής της διαδικτυακής υπηρεσίας, επικοινωνήστε στη διεύθυνσηprivacy.officer@acer.com. Εναλλακτικά, για συγκεκριμένες αιτήσεις ή/και ερωτήματα διαχείρισης τοπικών δεδομένων, επικοινωνήστε με την οντότητα της Acer στην οποία υποβάλατε τα στοιχεία σας. Οι διευθύνσεις επικοινωνίας για κάθε οντότητα της Acer είναι διαθέσιμες στη διεύθυνσηwww.acer-group.com/ag/en/TW/content/office-list. Για την παροχή ταχύτερης εξυπηρέτησης, υποβάλετε ερωτήσεις στα Αγγλικά, εάν είναι δυνατό, και ενημερώστε μας για τη χώρα από την οποία επικοινωνείτε μαζί μας, μαζί με μια περιγραφή της κατασκευής και του μοντέλου των προϊόντων σας (κατά περίπτωση).

8. Πώς να εξαιρεθείτε από τη χρήση των προσωπικών σας πληροφοριών:

Μπορείτε, ανά πάσα στιγμή, να εξαιρεθείτε από τη χρήση των προσωπικών σας πληροφοριών, επικοινωνώντας μαζί μας μέσω των παρακάτω στοιχείων επικοινωνίας. Θα αφαιρέσουμε τις προσωπικές σας πληροφορίες από τις βάσεις δεδομένων μας το συντομότερο δυνατό.

9. Συνδεδεμένοι ιστότοποι:

Άλλοι ιστότοποι που συνδέονται με/μέσα από τον ιστότοπό μας ενδέχεται να έχουν διαφορετικές πολιτικές. Σας συνιστούμε να διαβάσετε τις πολιτικές προστασίας προσωπικών δεδομένων που δημοσιεύονται σε αυτούς τους ιστοτόπους για λεπτομέρειες. Η Acer δεν εγγυάται ούτε υποστηρίζει οποιεσδήποτε πληροφορίες ή πολιτικές που παρέχονται σε οποιονδήποτε ιστότοπο που δεν ελέγχεται από την Acer.
