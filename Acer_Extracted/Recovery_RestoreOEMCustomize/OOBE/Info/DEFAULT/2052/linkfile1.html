2. 我们收集哪些信息以及如何收集：

我们收集各种信息，以帮助我们更好地为客户和网站访问者提供服务：

• 互联网协议 (IP) 地址。对于访问或使用我们的服务或浏览我们网站的每位访问者，我们的网络服务器会自动识别并存储访问者的 IP 地址（访问者计算机的互联网协议地址），以确定正在访问我们网站的哪些区域及每位访问者在那里停留的时间。这些信息有助于我们通过服务和网站了解访问者的流量模式，以便我们可以提高网站的整体可用性。您的计算机会在每次登录时自动提供此信息，除非您已激活某些浏览器中可用的屏蔽技术。

• 设备信息。我们可能会收集特定于设备的信息，例如您的硬件型号、操作系统版本和唯一设备标识符。此外，我们还可能使用浏览器 Web 存储（包括 HTML 5）和应用程序数据缓存等机制在您的设备上本地收集和存储信息（包括个人信息）。

• Cookie。我们将根据适用的数据保护法规使用 Cookie。

• 个人信息。虽然我们通常不要求您向我们提供个人身份信息以便浏览、访问或使用我们的网站，但有时我们可能会要求您向我们提供与个人身份信息相关的个人信息，例如您的电子邮件地址、姓名、邮寄地址、电话号码、信用卡号、国家/地区、语言、人口统计信息或永久标识符（如用户名和密码）。通过向我们提供此信息，您可以创建 AcerID、进行购买、注册产品、获得支持、与我们通信、参与在线调查、订阅邮件列表简报，或者注册特殊购买折扣。我们不要求您提供敏感的个人信息（例如有关您的健康、政治或宗教信仰、涉嫌或记录在案的刑事犯罪或性取向方面的信息），并要求您不要向我们提供此类信息。您自愿向我们提供的任何此类信息将仅根据本《隐私政策》使用。

• AcerID。为了允许您创建 AcerID，我们需要收集和处理以下部分或全部信息：电子邮件地址、姓名、姓氏、社交登录 ID（如 Facebook ID 和 Google ID）、国家/地区和语言。此信息量最少，因此是必需的，以便允许您创建 AcerID。创建 AcerID 后，您还可以通过提供更多信息来完善您的个人资料，例如您的电话号码、地址、宏碁产品注册所需的信息等。此外，您还可以通过选择接收宏碁简报和其他通信来管理您的个人资料，同时您可以随时有效取消相关订阅。

宏碁简报。您也可以选择在不创建 AcerID 的情况下接收我们的简报。为此，我们需要收集和处理的唯一信息是您的电子邮件地址、国家/地区和语言，而且这是向您发送我们的简报所必需的信息。
