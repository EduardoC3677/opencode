ข้อกำหนดในการใช้งานสำหรับโปรแกรมปรับปรุงประสบการณ์ผู้ใช้ของ Acer (Acer User Experience Improvement Program - UEIP)

โปรแกรมปรับปรุงประสบการณ์ผู้ใช้ของ Acer (เรียกสั้นๆ ว่า UEIP) ถูกออกแบบมาเพื่อเก็บข้อมูลของผู้ใช้โดยอัตโนมัติโดยตรงจากผู้ใช้ผลิตภัณฑ์ Acer จำนวนมาก เราจะใช้ข้อมูลดังกล่าวในการปรับปรุงผลิตภัณฑ์ของเรา เอกสารฉบับนี้อธิบายว่าเราเก็บข้อมูลใดบ้างภายใต้ UEIP ของ Acer และเราจะใช้ข้อมูลดังกล่าวอย่างไร

การเก็บและการใช้งานข้อมูล

ทันทีที่คุณให้ความยินยอมเข้าร่วมโปรแกรม ข้อมูลเกี่ยวกับอุปกรณ์ของคุณและวิธีการใช้อุปกรณ์เหล่านั้นจะถูกเก็บเป็นระยะ (สูงสุดวันละหนึ่งครั้ง) และถูกส่งไปที่ Acer ซึ่งเราจะใช้ข้อมูลดังกล่าวในการปรับปรุงผลิตภัณฑ์และบริการของเรา การส่งข้อมูลนี้อาจมีค่าบริการถ่ายโอนข้อมูล และ/หรือ ค่าบริการเชื่อมต่อ ทั้งนี้ขึ้นอยู่กับบริการเครือข่ายไร้สายของคุณ ข้อมูลดังกล่าวจะรวมถึงข้อมูลดังต่อไปนี้ในขอบเขตที่สามารถบังคับใช้:

• หมายเลขซีเรียลของอุปกรณ์, ชื่อรุ่นอุปกรณ์, ประเทศ, วันที่และเวลาเปิดใช้งานครั้งแรก, เวอร์ชัน UEIP, เวอร์ชันและชื่อระบบปฏิบัติการ, ข้อมูลฮาร์ดแวร์

• แอปพลิเคชันที่ปักหมุดบนแถบงาน

• การติดตั้ง/ยกเลิกการติดตั้งแอป

• ระบบ (ล่ม, เปิด/ปิด/S3/S4)

• สวิตช์นิรภัย

• การป้อนข้อมูลของผู้ใช้ (เมาส์/สไตลัส/ทัชสกรีน/ปากกา/ลายนิ้วมือ)

• เสียง (ปิดเสียง, ระดับเสียง)

• จอแสดงผล (เปิด/ปิด/หรี่)

• ที่อยู่ MAC ของจุดเข้าใช้งาน

• สถานะเครือข่าย (สถานะ/ความเร็วการเชื่อมต่อ)

• การใช้ดิสก์ (ขนาด, การใช้งาน)

• การใช้พลังงาน (ระดับแบตเตอรี่, โหมด AC/DC)

• การใช้แอปพลิเคชัน (เดสก์ท็อป/เมโทร)

• โหมดอุปกรณ์

• เครือข่าย (DNS)

• การใช้พอร์ต USB

• Wi-Fi (ประเภท/ความแรง/ช่องสัญญาณ)

• ฮาร์ดดิสก์ไดร์ฟ S.M.A.R.T.

• SD การ์ด

• การเปลี่ยนเป็นเต็มหน้าจอ

• ออปติคอลดิสก์ไดร์ฟ (ค่าเริ่มต้น/ใส่/ถอด)

• การป้อนค่าทางแป้นพิมพ์แบบจำกัด

• ความสว่างของจอแสดงผล

• วงจรแบตเตอรี่, กำลังสูงสุดของแบตเตอรี่, แผนการใช้พลังงาน

• แอป UWP

• ทางลัดเดสก์ท็อป

• โอเวอร์คล็อก

• ชื่อและประเภทของแอพบนอุปกรณ์ที่ใช้กล้องและ/หรือไมโครโฟนในตัว

• เวลาเริ่ม, เวลาสิ้นสุด และะยะเวลาสำหรับแอพที่เฉพาะเจาะจงที่ใช้กล้องและ/หรือไมโครโฟนในตัวสำหรับเซสชันเฉพาะ

รหัสเฉพาะสำหรับอุปกรณ์

Acer UEIP ใช้รหัสที่ไม่ซ้ำใครในระดับโลกเพื่อระบุอุปกรณ์ของคุณ (“รหัสเฉพาะสำหรับอุปกรณ์”) รหัสเฉพาะสำหรับอุปกรณ์ถูกสร้างและจัดเก็บแบบสุ่มในอุปกรณ์ของคุณ ข้อมูลเหล่านี้ไม่มีข้อมูลส่วนตัวใดๆ และไม่สามารถใช้ระบุตัวตนของคุณได้ Acer จะใช้ข้อมูลที่กล่าวด้านบนนี้เพื่อวัตถุประสงค์ในการปรับปรุงผลิตภัณฑ์ซึ่ง Acer จะมอบให้ในอนาคตและเพื่อตอบสนองความต้องการของผู้ใช้ให้ดีที่สุด ข้อมูลนี้จะไม่ถูกแบ่งปันหรือเปิดเผยกับหุ้นส่วนหรือบุคคลภายนอกใดๆ เว้นแต่จะได้รับมอบอำนาจตามกฎหมาย

หากฉันเลือกเข้าร่วม ฉันจะสามารถเลือกไม่รับในภายหลังได้หรือไม่?

ได้ คุณสามารถเลือกไม่รับโปรแกรมนี้ได้ตลอดเวลาโดยการเปลี่ยนแปลงการตั้งค่าผ่านส่วนติดต่อผู้ใช้ของโปรแกรมปรับปรุงประสบการณ์ผู้ใช้ของ Acer บนอุปกรณ์ Windows คุณสามารถหาการตั้งค่านี้ได้โดยการใช้ฟังก์ชัน ‘ค้นหา’ ของ Windows

การรักษาความปลอดภัยและความเป็นส่วนตัว

Acer มีความมุ่งมั่นในการปกป้องความเป็นส่วนตัวของคุณและการรักษาความปลอดภัยของข้อมูลที่ถูกเก็บรวบรวมโดยวิธีการหลากหลาย รวมทั้งการใช้เทคโนโลยีและกระบวนงานรักษาความปลอดภัย

หากต้องการข้อมูลเพิ่มเติมเกี่ยวกับวิธีการที่เราจัดการข้อมูลของลูกค้าและรักษาความปลอดภัยของข้อมูลดังกล่าว กรุณาไปที่ประกาศเกี่ยวกับสิทธิส่วนบุคคลด้านล่างนี้

ประกาศเกี่ยวกับสิทธิส่วนบุคคล

โดยประกาศเกี่ยวกับสิทธิส่วนบุคคลนี้ (ต่อไปนี้จะเรียกว่า “ประกาศเกี่ยวกับสิทธิส่วนบุคคล”) Acer Inc. (ต่อไปนี้จะเรียกว่า “Acer”, “บริษัท”, “เรา” หรือ “ทางเรา”) ที่มีสำนักงานจดทะเบียนที่ 8F, No. 88, Section 1, Xin Tai 5th Road, Xizhi, New Taipei City 221, Taiwan, R.O.C. ทำหน้าที่เป็นผู้ควบคุมข้อมูล แจ้งให้คุณทราบเกี่ยวกับหลักปฏิบัติที่เกี่ยวกับการประมวลผลข้อมูลที่บริษัทจะเป็นผู้ดำเนินการเมื่อคุณลงทะเบียนกับ UEIP

1. วัตถุประสงค์ของการประมวลผลข้อมูล

ทันทีที่คุณให้ความยินยอมเข้าร่วมโปรแกรม ข้อมูลเกี่ยวกับอุปกรณ์ของคุณและวิธีการใช้อุปกรณ์เหล่านั้นจะถูกเก็บเป็นระยะ (สูงสุดวันละหนึ่งครั้ง) และถูกส่งไปที่ Acer ซึ่งเราจะใช้ข้อมูลดังกล่าวในการปรับปรุงผลิตภัณฑ์และบริการของเรา วัตถุประสงค์ของการประมวลผลข้อมูลดังกล่าวทำเพื่อปรับปรุงคุณภาพและประสิทธิภาพการทำงานของอุปกรณ์ของเรามากยิ่งขึ้น เพื่อเป็นการรับประกันคุณสมบัติที่ดีที่สุดในอนาคตอันใกล้ ข้อกำหนดในการใช้งานรวมถึงรายการโดยละเอียดของข้อมูลที่ Acer จะประมวลผลในบริบทของ UEIP

ข้อมูลดังกล่าวจะถูกเก็บรวบรวมโดยอัตโนมัติและ Acer จะไม่สร้างโปรไฟล์ที่เฉพาะเจาะจงของพฤติกรรมในการใช้อุปกรณ์ของคุณ

2. การสื่อสารข้อมูลส่วนตัว

ข้อมูลที่ถูกเก็บรวบรวมจากอุปกรณ์ของคุณจะไม่ถูกแบ่งปันหรือเปิดเผยกับหุ้นส่วนหรือบุคคลภายนอกใดๆ เว้นแต่จะได้รับมอบอำนาจตามกฎหมายที่บังคับใช้

3. การถ่ายโอนข้อมูลส่วนตัว

ข้อมูลส่วนตัวอาจได้รับการถ่ายโอนไปที่ไต้หวัน ซึ่งเป็นที่ตั้งสำนักงานใหญ่ของบริษัท

4. การเก็บรักษาข้อมูลส่วนตัว

ข้อมูลจะได้รับการจัดเก็บเป็นเวลาไม่นานเกินความจำเป็นในการติดตามวัตถุประสงค์ของการประมวลผลข้อมูลที่อธิบายด้านบนนี้

5. สิทธิของผู้ใช้

คุณสามารถใช้สิทธิของคุณตามกฎหมายที่บังคับใช้ได้ตลอดเวลา ซึ่งรวมถึงสิทธิในการรับข้อมูลว่ามีข้อมูลส่วนตัวที่เกี่ยวกับคุณอยู่ในแฟ้มของ Acer หรือไม่ คุณสามารถขอทราบเนื้อหาและแหล่งข้อมูล ตรวจสอบความถูกต้องหรือขอแก้ไขหรือยกเลิกข้อมูลดังกล่าวได้ คุณสามารถเลือกไม่รับ UEIP ได้ตลอดเวลาโดยการเปลี่ยนแปลงการตั้งค่าผ่านส่วนติดต่อผู้ใช้ของโปรแกรมปรับปรุงประสบการณ์ผู้ใช้ของ Acer

หากมีคำถามหรือข้อคิดเห็นใดๆ เกี่ยวกับประกาศเกี่ยวกับสิทธิส่วนบุคคลนี้หรือเว็บไซต์ของเรา กรุณาติดต่อเราที่: privacy_officer@acer.com เพื่อให้เราสามารถมอบบริการที่รวดเร็วให้กับคุณได้ กรุณาส่งคำถามใดๆ เป็นภาษาอังกฤษหากสามารถทำได้ และโปรดแจ้งประเทศที่คุณใช้ติดต่อเรา ตลอดจนคำอธิบายยี่ห้อและรุ่นของผลิตภัณฑ์ของคุณ (ถ้ามี)
